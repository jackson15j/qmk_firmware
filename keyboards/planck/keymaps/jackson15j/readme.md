Jackson15j Planck
=================

See: [/users/jackson15j/], for layout and generic code.

Build
-----

From the root of the QMK folder:

```bash
make plank/rev6:jackson15j:dfu-util
```

Need to press the `RESET` button on base of the board to put it into Bootloader
mode.
